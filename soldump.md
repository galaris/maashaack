## Introduction ##

**soldump** is a command-line tool made with [redtamarin](http://code.google.com/p/redtamarin/).

It allows to dump the content of `*.sol` file or [Local Shared Object](http://en.wikipedia.org/wiki/Local_Shared_Object).
